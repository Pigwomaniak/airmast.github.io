magnification
=====
Set magnification

* **POST**

  * **Camera models**
    * Xt

  * **URL**  
    `magnification/<x>`
    
  * **URL Params**  
    * `x : Float`  
    
      Model Xt: magnify x times
      
  * **Command example:**
    * `/api/v1/magnification/1.5`
    * ```javascript
      {
        "success": true
      }
      ```
