zenmuse
=======

Choose Zenmuse camera
---------------------

.. class:: request-table-3

+--------------+-----------------------------------------+-------------+
| Request Type |                   URL                   | DJI Zenmuse |
+==============+=========================================+=============+
| POST         | **/api/v1/zenmuse/**\<zenmuse_option_\> | Any         |
+--------------+-----------------------------------------+-------------+

zenmuse_option
~~~~~~~~~~~~~~

+------------+-------------------------+
|   Option   |       Description       |
+============+=========================+
| **detect** | Auto detect camera type |
+------------+-------------------------+
| **x3**     | X3 camera               |
+------------+-------------------------+
| **x4s**    | X4S camera              |
+------------+-------------------------+
| **x5**     | X5 camera               |
+------------+-------------------------+
| **x5r**    | X5R camera              |
+------------+-------------------------+
| **x5s**    | X5S camera              |
+------------+-------------------------+
| **xt**     | XT camera               |
+------------+-------------------------+
| **z3**     | Z3 camera               |
+------------+-------------------------+
| **z30**    | Z30 camera              |
+------------+-------------------------+

Sample Request
~~~~~~~~~~~~~~

.. code:: http

    POST http://localhost:8123/api/v1/zenmuse/x3

Sample Response
~~~~~~~~~~~~~~~

Status code: **200**

.. code:: javascript

    {
      "success": true
    }
