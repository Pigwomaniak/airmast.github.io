gain_mode
=====
Set gain mode

* **POST**

  * **Camera models**
    * Xt

  * **URL**  
    `gain_mode/<mode>`
    
  * **URL Params**  
    * `mode : String`  
    
      Model Xt:
      
      `auto`  
      `low`  
      `hight`
      
  * **Command example:**
    * `/api/v1/gain_mode/auto`
    * ```javascript
      {
        "success": true
      }
      ```
