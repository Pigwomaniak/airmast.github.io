mode
=====
Camera mode

* **POST**

  * **URL**  
    `mode/<mode>`
    
  * **URL Params**  
    * `mode : String`  
    
      `photo` &mdash; Photo  
      `video` &mdash; Video  
      `playback` &mdash; Playback
      
  * **Command example:**
    * `/api/v1/mode/photo`
    * ```javascript
      {
        "success": true
      }
      ```
