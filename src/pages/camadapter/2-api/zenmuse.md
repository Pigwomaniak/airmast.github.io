zenmuse
=====
Set used Zenmuse camera

* **POST**

  * **URL**  
    `zenmuse/<model>`
    
  * **URL Params**  
    * `model : String` 
  
      `x3`  
      `x4s`  
      `x5`  
      `x5r`  
      `x5s`  
      `xt`  
      `z3`  
      `z30`  
      `detect` &mdash; Auto detect camera type  
      
  * **Command example:**
    * `/api/v1/zenmuse/x3`
    * ```javascript
      {
        "success": true
      }
      ```
