shutter
=====
Set shutter value

* **POST**

  * **Camera models**
    * X3

  * **URL**  
    `shutter/<numerator>d<denominator>`
    
  * **URL Params**  
    Model X3:
    
    * `numerator : Int`  
    * `denominator : Int`  
      
  * **Command example:**
    * `/api/v1/shutter/1d125`
    * ```javascript
      {
        "success": true
      }
      ```
