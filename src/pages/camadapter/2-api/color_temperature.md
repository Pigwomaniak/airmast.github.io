color_temperature
=====
Set color temperature value

* **POST**

  * **Camera models**
    * X3

  * **URL**  
    `color_temperature/<temp>`
    
  * **URL Params**  
    * `temp : UInt`  
      
       Model X3: set color temperature in Kelvins
      
  * **Command example:**
    * `/api/v1/color_temperature/2700`
    * ```javascript
      {
        "success": true
      }
      ```
