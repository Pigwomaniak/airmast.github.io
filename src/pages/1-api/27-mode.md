mode
=====
Camera mode

* **POST**

  * **URL**  
    `mode/<mode>`
    
  * **URL Params**  
    * `mode : String`  
    
      `photo` - Photo  
      `video` - Video  
      `playback` - Playback
      
  * **Command example:**
    * `/api/v1/mode/photo`
    * ```javascript
      {
        "success": true
      }
      ```
