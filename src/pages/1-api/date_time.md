date_time
=====
Set date and time

* **POST**

  * **URL**  
    `date_time/<dt>`
    
  * **URL Params**  
    * `dt : String`  
       Set date and time ("yyyy-MM-ddThh:mm:ss", T &mdash; delimiter)
      
  * **Command example:**
    * `/api/v1/date_time/2017-08-05T11:30:15`
    * ```javascript
      {
        "success": true
      }
      ```
