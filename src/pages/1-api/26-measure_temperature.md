measure_temperature
=====
Enable or disable temperature measurement

* **POST**

  * **Camera models**
    * Xt

  * **URL**  
    `measure_temperature/<enable>`
    
  * **URL Params**  
    * `enable : String`  
    
      Model Xt: `on`, `off`
      
  * **Command example:**
    * `/api/v1/measure_temperature/on`
    * ```javascript
      {
        "success": true
      }
      ```
