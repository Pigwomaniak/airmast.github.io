isotherm
=====
Enable or disable isotherm

* **POST**

  * **Camera models**
    * Xt

  * **URL**  
    `isotherm/<enable>`
    
  * **URL Params**  
    * `enable : String`  
    
      Model Xt:
      
      `on`, `off` - Enable or disable isotherm
      
  * **Command example:**
    * `/api/v1/isotherm/on`
    * ```javascript
      {
        "success": true
      }
      ```
