color
=====
Color mode

* **POST**

  * **Camera models**
    * X3
    * Xt

  * **URL**  
    `color/<color>`
    
  * **URL Params**  
    * `color : String`  
      
       Model X3:
    
      `none` - None (default after power up)  
      `art` - Art  
      `bw` — Black White  
      `d-cinelike` — D-Cinelike  
      `vivid` — Vivid  
      `beach` — Beach  
      `dream` — Dream  
      `classic` — Classic  
      `jugo` — Jugo  
      `d-log` — D-Log  
      
      Model Xt:
      
      `whitehot`  
      `blackhot`  
      `fusion`  
      `rainbow`  
      `ironbow1`  
      `icefire`  
      `sepia`  
      `globow`  
      `ironbow2`  
      `color1`  
      `color2`  
      `rain`  
      `redhot`  
      `greenhot`
