color_system
=====
Set NTSC or PAL color system

* **POST**

  * **Camera models**
    * Xt

  * **URL**  
    `color_system/<system>`
    
  * **URL Params**  
    * `system : String`  
      
       Model Xt:
    
      `ntsc`  
      `pal`
      
  * **Command example:**
    * `/api/v1/color_system/pal`
    * ```javascript
      {
        "success": true
      }
      ```
