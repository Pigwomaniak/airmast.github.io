scene
=====
Set camera's scene type

* **POST**

  * **Camera models**
    * Xt

  * **URL**  
    `scene/<type>`
    
  * **URL Params**  
    * `type : String`  
    
      Model Xt:
      
      `linear`  
      `default`  
      `seasky`  
      `outdoor`  
      `indoor`  
      `manual`  
      `user1`  
      `user2`  
      `user3`  
      
  * **Command example:**
    * `/api/v1/scene/linear`
    * ```javascript
      {
        "success": true
      }
      ```
