video_format
=====
Set video format

* **POST**

  * **Camera models**
    * X3

  * **URL**  
    `video_format/<format>`
    
  * **URL Params**  
    * `format : String`
  
      Model X3: `mov`, `mp4`
      
  * **Command example:**
    * `/api/v1/video_format/mov`
    * ```javascript
      {
        "success": true
      }
      ```
