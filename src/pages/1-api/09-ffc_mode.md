ffc_mode
=====
Set flat-field correction mode

* **POST**

  * **Camera models**
    * Xt

  * **URL**  
    `ffc_mode/<mode>`
    
  * **URL Params**  
    * `mode : String`  
    
      Model Xt:
      
      `auto`  
      `manual`  
      
  * **Command example:**
    * `/api/v1/ffc_mode/auto`
    * ```javascript
      {
        "success": true
      }
      ```
