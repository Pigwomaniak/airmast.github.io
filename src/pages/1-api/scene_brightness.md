scene_brightness
=====
Set scene brightness

* **POST**

  * **Camera models**
    * Xt

  * **URL**  
    `scene_brightness/<brightness>`
    
  * **URL Params**  
    * `brightness : Integer`  
    
      Model Xt: `brightness` &mdash; integer from 0 to 16383
      
  * **Command example:**
    * `/api/v1/scene_brightness/7000`
    * ```javascript
      {
        "success": true
      }
      ```
