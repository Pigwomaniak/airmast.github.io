scene_contrast
=====
Set scene contrast

* **POST**

  * **Camera models**
    * Xt

  * **URL**  
    `scene_contrast/<contrast>`
    
  * **URL Params**  
    * `contrast : Integer`  
    
      Model Xt: `contrast` &mdash; contrast value from 0 to 255
      
  * **Command example:**
    * `/api/v1/scene_contrast/100`
    * ```javascript
      {
        "success": true
      }
      ```
