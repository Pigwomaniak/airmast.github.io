scene_ace
=====
Set active contrast enhancement

* **POST**

  * **Camera models**
    * Xt

  * **URL**  
    `scene_ace/<ace>`
    
  * **URL Params**  
    * `ace : Integer`  
    
      Model Xt: `ace` - integer from -8 to +8
      
  * **Command example:**
    * `/api/v1/scene_ace/2`
    * ```javascript
      {
        "success": true
      }
      ```
