scene_sso
=====
Set smart scene optimization

* **POST**

  * **Camera models**
    * Xt

  * **URL**  
    `scene_sso/<sso>`
    
  * **URL Params**  
    * `sso : Integer`  
    
      Model Xt: `sso` &mdash; SSO is a percent from 0 to 100
      
  * **Command example:**
    * `/api/v1/scene_sso/75`
    * ```javascript
      {
        "success": true
      }
      ```
