roi
=====
Set camera's region of interest

* **POST**

  * **Camera models**
    * Xt

  * **URL**  
    `roi/<roi>`
    
  * **URL Params**  
    * `roi : String`  
    
      Model Xt: `full`, `skyex33`, `skyex50`
      
  * **Command example:**
    * `/api/v1/roi/full`
    * ```javascript
      {
        "success": true
      }
      ```
