isotherm

Enable or disable isotherm

Request Type URL DJI Zenmuse
POST /api/v1/isotherm/<isotherm_action> XT

isotherm_action

Option Description
off Disable isotherm
on Enable isotherm

Sample Request

POST http://localhost:8123/api/v1/isotherm/on

Sample Response

Status code: 200

{
  "success": true
}
