color_system

Set NTSC/PAL color system

Request Type URL DJI Zenmuse
POST /api/v1/color_system/<color_system_value> X3, XT, Z3, Z30

color_system_value

Option Description
ntsc Set color system to NTSC
pal Set color system to PAL

Sample Request

POST http://localhost:8123/api/v1/color_system/pal

Sample Response

Status code: 200

{
  "success": true
}
